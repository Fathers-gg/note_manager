# note_manager
 1. Файл greetengs.py содержит в себе переменные и print(),
который выводит эти переменные на экран. так же содержит импорт бибилиотеки datetime.
2. Файл addd_input.py содержит в себе переменные, но на это раз на экран выводится 
введенное пользователем значение переменной с помощью команды input
3. Файл ADD_LIST.py содержит в себе заголовки, которые затем были объединены в список
titles=[title_1, title_2, title_3]. 
4. Файл final.py содержит в себе содержимое предыдущих файлов в более читаемом виде и с 
исправленной датой. добавлен словарь.
5. Файл date_changer.py содержит в себе ввод даты в формате дд-мм-ггг, а вывод происходит в формате дд-мм.
6. Файл изменения.ру содержит в себе усложненный мной код для гибкости программы и возможности 
неограниченного кол-ва заметок и заголовков. были использованы: main(), if-else, .append, for, 
while, True, try-exept. к каждой строке кода дан комментарий для понимания
как это работает и что происходит в коде.
